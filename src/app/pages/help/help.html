Help

Welcome to the help page for Join, your guide to using our kanban project management tool.

What is Join?

Join is a kanban-based project management tool designed to help users organize tasks, contacts and workflows.

How to use it

  1. Exploring the Board

    Use the board to see your tasks grouped by status.

  2. Creating Contacts

    Add contacts so they can be assigned to tasks.

  3. Adding Cards

    Create new tasks with title, description, due date, priority and category.

  4. Moving Cards

    Move tasks between the different board columns as your work progresses.

  5. Deleting Cards

    Once a task is completed, you can either move it to the "Done" list or delete it. Deleting a card will permanently remove it from the board. Please exercise caution when deleting cards, as this action is irreversible. Remember that using Join effectively requires consistent updates from you and your team to ensure the board reflects the current state of your project. Have more questions about Join?
